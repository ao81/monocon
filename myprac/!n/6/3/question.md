# 電子回路組立部門 練習課題

## 課題名

複合センサー判定装置

## 使用する入力

```cpp
Di sw1(d3), sw2(d2), ph(d4), ts(d1);
Sok sok(a1);
Pr pr(a2);
Vr vr(a3);
```

この課題では測距センサー`Sok`は使用しない。

| オブジェクト | 接続機器      |
| ------ | --------- |
| `sw1`  | タクトスイッチ1  |
| `sw2`  | タクトスイッチ2  |
| `ph`   | フォトインタラプタ |
| `ts`   | トグルスイッチ   |
| `pr`   | 光センサー     |
| `vr`   | 可変抵抗器     |

## 課題概要

フォトインタラプタと光センサーの入力順序、入力時間、入力間隔を解析し、通過した物体の種類を判定する装置を作成しなさい。

フォトインタラプタはモーターとは連動しておらず、物体を手動で通過させて反応させるものとする。

---

## 1．測定許可

`ts`を測定許可用トグルスイッチとする。

### `ts`がOFFの場合

* 測定を行わない
* 測定途中のデータを破棄する
* 7セグメントLEDに`OFF`を表示する
* フルカラーLEDを消灯する
* ブザーを停止する

### `ts`がONの場合

* 新しい測定を受け付ける
* OFF中に発生した入力を引き継がない

---

## 2．判定基準の設定

可変抵抗器の値を50～500msへ変換し、判定基準時間とする。

```cpp
int limitTime = vr.to(50, 500);
```

設定値は7セグメントLEDに表示すること。

`sw1`を押すと設定値を確定し、測定を開始できる状態にすること。

確定後に可変抵抗器を動かしても、保存した設定値は変化しないこと。

`sw2`を押すと設定状態へ戻ること。

---

## 3．入力時間の測定

フォトインタラプタ`ph`について、遮光された瞬間から遮光が解除されるまでの時間を測定すること。

光センサー`pr`についても、反応開始から反応終了までの時間を測定すること。

それぞれ次の値を保存すること。

* フォトインタラプタの反応時間
* 光センサーの反応時間
* 先に反応したセンサー
* 2つのセンサーが反応を開始した時刻の差

5ms未満の反応はノイズとして無視すること。

---

## 4．物体の種類判定

1回の測定では、フォトインタラプタと光センサーがそれぞれ1回ずつ反応するものとする。

2つの入力結果から、次のように物体を判定すること。

### 種類A

* フォトインタラプタが先に反応する
* 2つの反応開始時刻の差が基準時間未満
* フォトインタラプタの反応時間が光センサーより短い

表示：

```text
A
```

### 種類B

* 光センサーが先に反応する
* 2つの反応開始時刻の差が基準時間未満
* 光センサーの反応時間がフォトインタラプタより短い

表示：

```text
b
```

### 種類C

* 2つの反応開始時刻の差が基準時間以上
* 先に反応したセンサーの種類は問わない

表示：

```text
C
```

### 種類D

* 2つのセンサーがほぼ同時に反応する
* 開始時刻の差が20ms以下
* 反応時間の差も20ms以下

表示：

```text
d
```

### 判定不能

上記のどれにも当てはまらない場合は判定不能とする。

表示：

```text
Err
```

---

## 5．判定の優先順位

種類Dの条件は、種類Aや種類Bの条件と同時に成立する可能性がある。

そのため、判定する順番を自分で考えること。

誤った順番で判定すると、本来種類Dである物体が種類Aまたは種類Bとして判定される。

判定条件が重複した場合でも、必ず一意に判定できる処理にすること。

---

## 6．測定の区切り

1つの物体について、両方のセンサーの反応が終了した時点で判定を行うこと。

片方だけが反応し、もう片方が2秒以内に反応しなかった場合はタイムアウトとする。

タイムアウト時は次の動作を行う。

* 7セグメントLEDに`Err`を表示する
* フルカラーLEDを赤色で点灯する
* 500Hzの音を200ms鳴らす
* 測定途中のデータを破棄する

タイムアウト後は、両方のセンサーが非反応状態になってから次の測定を受け付けること。

---

## 7．連続入力への対応

前の物体がセンサーから完全に離れる前に、次の物体が入力された場合は測定を開始してはならない。

次の条件をすべて満たしたときだけ、新しい測定を許可すること。

* フォトインタラプタが非反応
* 光センサーが非反応
* 両方が非反応になってから100ms以上経過している

これにより、1つの物体を複数回判定しないようにすること。

---

## 8．判定結果の記録

種類A、B、C、Dの判定回数をそれぞれ保存すること。

```cpp
int countA;
int countB;
int countC;
int countD;
```

`sw2`を押すたびに、次の表示を切り替えること。

1. 最後の判定結果
2. 種類Aの個数
3. 種類Bの個数
4. 種類Cの個数
5. 種類Dの個数
6. 判定不能の個数

表示モードを切り替えても、センサーの測定は継続すること。

---

## 9．連続判定の分析

直近5回分の判定結果を保存すること。

同じ種類が3回以上連続して判定された場合は、連続判定として通知すること。

通知時は次の動作を行う。

* 7セグメントLEDに連続回数を表示する
* フルカラーLEDを黄色で点滅させる
* 2000Hzの音を50ms鳴らす

異なる種類が判定されたら、連続回数を1へ戻すこと。

判定不能は連続回数に含めないこと。

---

## 10．センサー異常判定

次のいずれかが発生した場合は、センサー異常と判断すること。

* フォトインタラプタが5秒以上連続して反応している
* 光センサーが5秒以上連続して反応している
* 10回連続で片方のセンサーしか反応しない
* 反応開始と終了が短時間に繰り返され、5ms未満のノイズが1秒間に10回以上発生した

異常時は次の動作を行う。

* 測定を停止する
* 7セグメントLEDに`Err`を表示する
* フルカラーLEDを赤色で点滅させる
* 1000Hzと500Hzを交互に鳴らす

`sw1`と`sw2`を同時に1秒以上押すと、異常を解除すること。

解除時には、測定途中のデータとノイズ回数を初期化すること。

---

## 11．自動基準値設定

`sw1`を1秒以上長押しすると、自動設定モードへ入ること。

自動設定モードでは、その後に得られた5回分のセンサー開始時刻差を保存する。

5回分の値を小さい順に並べ、その中央値を新しい基準時間とすること。

例：

```text
120ms
125ms
119ms
400ms
123ms
```

中央値は123msとなる。

最大値400msは外れ値であるが、中央値を使うことで基準値への影響を抑えられる。

---

## 12．考察問題

1. 種類Dを種類Aや種類Bより先に判定すべき理由を説明しなさい。
2. 両センサーが非反応になった直後に次の測定を始めない理由を説明しなさい。
3. 平均値ではなく中央値を使用する利点を説明しなさい。
4. 片方のセンサーだけが反応した状態で無制限に待つと、どのような問題が発生するか説明しなさい。
5. センサーの開始時刻差と反応時間は、それぞれ物体のどのような特徴を表すと考えられるか説明しなさい。
6. タクトスイッチの長押しと通常押下を区別する方法を説明しなさい。
7. 測定中に`ts`がOFFになった場合、途中データを残してはいけない理由を説明しなさい。
8. 5ms未満の反応をすべて無視した場合、実際に非常に高速な物体が通過したときにどのような問題が起こるか説明しなさい。

---

## 制作条件

* `delay()`を使用しないこと
* 待機用の`while`文を使用しないこと
* タクトスイッチにはエッジ検出を使用すること
* トグルスイッチは現在のレベルを継続して確認すること
* センサー入力の開始時刻と終了時刻を個別に記録すること
* タイムアウトを実装すること
* 測定中もスイッチ操作と表示切り替えを受け付けること
* 配列の範囲外へアクセスしないこと
* `millis()`のオーバーフローに対応できる時間差計算を使用すること
