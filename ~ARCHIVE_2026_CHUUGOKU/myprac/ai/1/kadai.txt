問題1　DCモータと表示制御　　　配点10点
◇◇◇ a パターン（最大5点）◇◇◇
■ 初期状態　注意事項(7)に記載の通りとする。
■ 動作内容　タクトSWとフォトインタラプタを用いて、下記を満たすDCモータの制御プログラムを作成する。

タクトSWがON、かつフォトインタラプタが透過のとき、DCモータはCW方向に回転する。
タクトSWがON、かつフォトインタラプタが遮断されているとき、DCモータはCCW方向に回転する。
タクトSWがOFFのとき、DCモータは停止する(フォトインタラプタの状態に関わらず)。

タクトSWフォトインタラプタDCモータ備考ON透過CW小問(1)ON遮断CCW小問(2)OFF―停止小問(3)
◇◇◇ b パターン（最大10点）◇◇◇
■ 初期状態　注意事項(7)に記載の通りとする。
■ 動作内容　トグルSW、タクトSW、フォトインタラプタを用いて、下記を満たす制御プログラムを作成する。

トグルSWがOFFのとき、DCモータは停止し、7セグメントLEDは"00"を表示する。フルカラーLEDは消灯。
トグルSWがONのとき、下記の仕様を満たす:

(ア) フルカラーLED（青）を点灯する。
(イ) タクトSWがONの間、DCモータはCW方向に回転する。
(ウ) タクトSWがOFFの間、DCモータはCCW方向に回転する。
(エ) フォトインタラプタが透過→遮断に変わる度に、7セグメントLEDの値を+1カウントアップする(00→01→…→99→00)。
(オ) 7セグメントLEDの値が99→00に戻るタイミングで、フルカラーLED（青）を200msec消灯（その後再点灯）。


トグルSWを再度OFFにすると(1)の状態に戻る。カウント値はリセット。


問題2　フルカラーLEDによる信号機制御　　　配点15点
◇◇◇ a パターン（最大5点）◇◇◇
■ 初期状態　注意事項(7)に記載の通りとする。
■ 動作内容　トグルSWを用いて、下記を満たすフルカラーLEDの制御プログラムを作成する。

トグルSWがONの間、フルカラーLEDが以下の周期で繰り返し点灯する:
緑3秒 → 黄1秒 → 赤3秒 → 緑3秒 → …
※ 黄色は赤と緑のフルカラーLEDを同時点灯することで表現する。
トグルSWがOFFの間、すべてのフルカラーLEDを消灯する。

◇◇◇ b パターン（最大15点）◇◇◇
■ 初期状態　注意事項(7)に記載の通りとする。
■ 動作内容　トグルSWとタクトSWを用いて、下記を満たすフルカラーLEDと7セグメントLEDの制御プログラムを作成する。

トグルSWがONの間、フルカラーLEDが以下の周期で繰り返し点灯する:
緑5秒 → 緑点滅(0.5秒間隔で2回点滅)→ 黄2秒 → 赤5秒 → 緑5秒 → …
※ 黄色は赤と緑の同時点灯。点滅は緑LEDのみを点滅させる。
7セグメントLEDに、現在の状態の残り秒数を2桁で表示する。

緑5秒中: "05"→"04"→…→"01"（1秒ごとに減算）
緑点滅中: "00"を表示
黄2秒中: "02"→"01"
赤5秒中: "05"→…→"01"


緑信号点灯中（点滅中含む）にタクトSWを押下すると、即座に黄2秒の状態へ遷移する（歩行者ボタン的動作）。緑以外の状態でのタクトSW入力は無視する。
トグルSWをOFFにすると、すべてのフルカラーLEDを消灯し、7セグメントLEDを"00"に戻す。


問題3　ステッピングモータのアナログ制御　　　配点15点
◇◇◇ a パターン（最大5点）◇◇◇
■ 初期状態　注意事項(7)に記載の通りとする。
■ 動作内容　ジョイスティックを用いて、下記を満たすステッピングモータの制御プログラムを作成する。

ジョイスティックを上に倒すとステッピングモータはCW方向に回転する。
ジョイスティックが中心にあるとき、ステッピングモータは停止する。
ジョイスティックを下に倒すとステッピングモータはCCW方向に回転する。

ジョイスティックステッピングモータ上CW中心停止下CCW
◇◇◇ b パターン（最大15点）◇◇◇
■ 初期状態　注意事項(7)に記載の通りとする。
■ 動作内容　ジョイスティックとトグルSWとタクトSWを用いて、下記を満たすステッピングモータと7セグメントLEDの制御プログラムを作成する。

トグルSWがONの間、ジョイスティックのX軸方向の倒し具合でステッピングモータの回転速度と方向を5段階で制御する:

右に強く倒す: CW高速
右に少し倒す: CW低速
中心: 停止
左に少し倒す: CCW低速
左に強く倒す: CCW高速


ステッピングモータの累計回転量を7セグメントLEDで表現する。CW方向で90°回転するごとに+1、CCW方向で90°回転するごとに-1とし、現在の累計値を表示する。表示範囲は -9 〜 +9。範囲を超えた回転入力に対しては、ステッピングモータも停止する。

正の値: 1桁目に数字、2桁目消灯
負の値: 1桁目に数字、2桁目に"-"
0: "00"


トグルSWがOFFの間、ジョイスティック操作を無効にしてステッピングモータを停止する。表示は維持する。
トグルSWがOFFの状態でタクトSWを押下している間、ステッピングモータは最も近い経路で初期位置(0°、表示"00")に向けて低速で戻る。タクトSWを離すと停止する。0°に到達したら自動停止する。

トグルSWジョイスティックステッピングモータON右強CW高速ON右弱CW低速―中心停止ON左弱CCW低速ON左強CCW高速OFF―停止(タクトSW押下時のみ初期位置へ低速復帰)

問題4　ブザーによる音階再生　　　配点20点
◇◇◇ a パターン（最大10点）◇◇◇
■ 初期状態　注意事項(7)に記載の通りとする。
■ 動作内容　トグルSWを用いて、下記を満たすブザーの制御プログラムを作成する。

トグルSWがONの間、ブザーから以下の音階を400msec間隔で繰り返し再生する:
C4 → D4 → E4 → F4 → G4 → A4 → B4 → C5 → C4 → …
7セグメントLEDに現在再生中の音を以下の通り表示する:
C4="01"、D4="02"、E4="03"、F4="04"、G4="05"、A4="06"、B4="07"、C5="08"
トグルSWがOFFの時、ブザーは停止し、7セグメントLEDは"00"を表示する。

◇◇◇ b パターン（最大20点）◇◇◇
■ 初期状態　注意事項(7)に記載の通りとする。
■ 動作内容　トグルSW、タクトSW、フォトインタラプタを用いて、下記を満たすブザーと7セグメントLEDの制御プログラムを作成する。

トグルSWがONの間、ブザーから以下の音階を標準400msecずつ順番に再生する。終端まで進むと折り返して繰り返す:
C4 → D4 → E4 → F4 → G4 → A4 → B4 → C5 → B4 → A4 → G4 → F4 → E4 → D4 → C4 → D4 → …
7セグメントLEDに現在再生中の音階を表示する:
C4="01"、D4="02"、E4="03"、F4="04"、G4="05"、A4="06"、B4="07"、C5="08"
フォトインタラプタが遮断されている間、再生速度が2倍になる(各音200msec)。透過に戻れば400msecに復帰する。
タクトSWを押下している間、現在の音階を持続して鳴らし続ける(再生は一時停止)。タクトSWを離すと、保持していた次の音から再生を再開する。7セグメントLEDの表示も保持する。
トグルSWをOFFにすると、ブザーは停止し、7セグメントLEDは"00"に戻る。再度ONにするとC4から再生が始まる。

トグルSWフォトインタラプタタクトSWブザー備考ON透過OFF400msec間隔で再生通常再生ON遮断OFF200msec間隔で再生倍速再生ON―ON現在音を持続一時停止OFF――停止表示も"00"

問題5　順序記憶ゲーム　　　配点40点
■ 概要
コンピュータが提示する**色と音の順序(シーケンス)**を覚え、プレイヤーがジョイスティックとタクトSWで再現する記憶ゲーム。正しく再現できるとラウンドが進み、シーケンスが1つずつ長くなっていく。
■ システム構成図

入力回路: トグルSW、タクトSW、ジョイスティック
出力回路: 7セグメントLED、フルカラーLED、ブザー、ステッピングモータ

■ 初期状態
注意事項(7)に記載の通りとする。ただし7セグメントLEDは全消灯とする。
■ 動作内容
(1) ゲーム状態の管理（配点5点）
トグルSWでゲーム状態を管理する。
トグルSWゲーム状態フルカラーLEDONゲーム中状態緑点灯OFFゲーム初期状態消灯
ゲーム中状態でタクトSWを押下するとゲームが開始される。
ゲーム開始後にトグルSWをOFFにした場合は、強制的にゲーム初期状態へ戻り、すべての出力は初期化される。
(2) シーケンスの提示（配点10点）
ゲーム開始後、ラウンド1から始まる。
ラウンドN（N=1, 2, 3, …, 9）では、N個の「色」を順番に提示する。各色は次の3種類からランダムに選ばれる:
色フルカラーLED音階赤赤点灯C4緑緑点灯E4青青点灯G4
各色の点灯+発音時間は500msec、間の無音/消灯間隔は200msecとする。
シーケンス提示中、7セグメントLEDの2桁目に現在のラウンド数、1桁目に提示中の色の通し番号(1, 2, 3, …)を表示する(例: ラウンド3の2個目を提示中なら"32")。
(3) プレイヤーによる再現（配点10点）
シーケンス提示完了後、プレイヤーはジョイスティックで色を選択し、タクトSWで確定する。
ジョイスティック選択する色左赤真ん中(中心)緑右青

入力中、ジョイスティックの位置に対応する色のフルカラーLEDを200msec点灯/200msec消灯で点滅させ、現在の選択色をユーザに示す。
タクトSWを押下した瞬間に対応する色を500msec点灯し、同時に音(C4/E4/G4)を鳴らす(確定動作)。
確定する度に7セグメントLEDの1桁目をN → N-1 → … → 1 → 0とカウントダウン表示し、残り入力数を示す。
N個入力したら自動的に判定((4))へ進む。

(4) 判定とラウンド進行（配点10点）
入力したシーケンスとコンピュータの提示シーケンスを比較する:

全て一致: 正解ファンファーレ(C5を100msec → E4を100msec → G4を100msec → C5を300msec)を鳴らし、ラウンドを+1して(2)に戻る。
一つでも不一致: 不正解音(B3を300msec → B3を300msec、間に100msecの無音)を鳴らし、ゲーム終了。
ラウンド9を達成したらゲームクリア。勝利のメロディ(C4 → E4 → G4 → C5、各300msec)を3回繰り返す。

ゲーム終了/クリア後、トグルSWをOFFにするまで7セグメントLEDに最終結果を表示し続ける:

失敗時: 失敗したラウンドの 直前にクリアしたラウンド数(例: ラウンド3で失敗 → "02"。ラウンド1で失敗 → "00")
クリア時: "09"

ゲーム終了/クリア後はトグルSWをOFF→ONにするまでじゃんけん(タクトSW)は無効。
(5) ステッピングモータによる進捗表示（配点5点）
ラウンドが進むごとに、ステッピングモータがCW方向に20°回転する。

ラウンド1開始時: 0°（初期位置）
ラウンド2開始時: 20°
…
ラウンド9開始時: 160°
ゲーム終了時(失敗)、ステッピングモータは最も近い経路で初期位置(0°)に戻る。
ゲームクリア時、ステッピングモータは360°のCW回転を1回行ってから0°で停止する。
トグルSWをOFFにしたら、ステッピングモータも初期位置(0°)に戻る。
